如何获取Copilot Cookie

🎯 简单3步获取Cookie：

1
访问Copilot网站
🚀 生产环境 🧪 测试环境 🌐 备用环境
2
打开开发者工具

按 F12 或右键选择"检查" → 点击 Network 标签页

3
复制Cookie

在网站上发送一个对话 → 在Network中找到API请求 → Request Headers中复制完整的 Cookie 值

⚠️ 注意事项：

  • Cookie包含认证信息，请妥善保管
  • Cookie可能有时效性，失效后需要重新获取
  • 不同环境的Cookie不能混用
  • 配置任意一个Cookie即可启用对应环境的2个模型

🔍 示例Cookie格式：

passport=MTc1NjI4NzM4N3x...; __TRACKER__USER_INFO__={"userUniqueId":"..."};
